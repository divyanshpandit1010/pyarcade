# 🎮 PyArcade — Python Gaming Platform

Python algorithms running live in the browser.

🔗 **Live Demo:** https://divyanshpandit1010.github.io/pyarcade

- ⭕ Tic Tac Toe (Minimax AI)
- 🧠 Python Quiz

- 🃏 Memory Flip
- 🏓 Pong
